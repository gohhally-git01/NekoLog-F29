# ネコログF29

ネコログF29 は、ログファイルの整理・抽出・可視化を目的とした Python ツールです。
GUI を持たないシンプルな構成で、現場での実用性と軽量性を重視しています。

## 特徴
- 指定フォルダ内のログファイルを自動整理
- 必要な情報だけを抽出してまとめる
- シンプルな設定で動作
- Python 標準ライブラリ中心で依存が少ない

## 必要環境
- Python 3.x
- OS: Windows / macOS / Linux
